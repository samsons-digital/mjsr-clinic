📦 Bulk Stock Upload

Upload a CSV file with medicine names/SNOs and quantities to add stock in bulk

The file must have columns for Name (or SNO) and Qty (or Quantity / Balance / Received). The system will match against current medicines by name or SNO, preview the matches, and let you confirm before writing anything.

Auto-sync mode: Each medicine balance will be adjusted to exactly match the quantity in your file. If current balance is higher it will be corrected down. If lower it will be topped up. No manual zeroing needed.
CSV format example:
Name,Qty
Tab Panadol 500mg,200
Inj Transim,50

You can also use columns: SNO, Particulars, Received, New Stock, Balance — the system auto-detects common column names.
For Excel files: save as CSV first via File > Save As > CSV.
